# PHPBox — Universal PHP Development Environment Manager

## Vision

PHPBox is a cross-platform CLI tool that allows developers to:

* Create new PHP projects
* Run existing PHP projects
* Manage PHP versions
* Manage databases
* Manage web servers
* Manage PHP extensions
* Manage Docker environments
* Manage framework-specific commands

without installing any of the following on the host machine:

* PHP
* Composer
* Apache
* Nginx
* MySQL
* MariaDB
* PostgreSQL
* XAMPP
* WAMP
* MAMP
* Laragon

The only required dependency is:

* Docker Desktop (Windows / Mac)
* Docker Engine + Docker Compose (Linux)

The architecture uses Docker containers for complete isolation and portability. A modern PHP stack is based on separate containers for the web server, PHP runtime, and database services.

# Configuration File (`phpbox.yml`)

`phpbox.yml` is the single source of truth for a project's environment. It is generated by `phpbox init` / `phpbox create` and can be edited by hand. The `.phpbox/` artifacts (docker-compose, Dockerfiles, vhosts) are regenerated from it.

```yaml
# phpbox.yml
name: blog
framework: laravel
document_root: /public

php:
  version: "8.4"
  extensions:
    - redis
    - gd
    - zip
    - intl
  ini:
    memory_limit: 256M
    upload_max_filesize: 64M

composer:
  version: latest

server:
  type: nginx          # nginx | apache | litespeed | caddy

database:
  engine: mariadb      # mariadb | mysql | postgres | sqlite
  version: "11"
  name: blog
  user: blog
  password: secret

services:
  redis: true
  mailpit: true
  meilisearch: false
  elasticsearch: false
  phpmyadmin: true

ssl:
  enabled: false
  host: blog.localhost

ports:
  http: 8080
  https: 8443
  database: 3306
  redis: 6379
  mailpit: 8025
```

# Plugin Architecture

```txt
plugins/
│
├── laravel.py
├── symfony.py
├── wordpress.py
├── yii.py
├── cakephp.py
├── codeigniter.py
└── corephp.py
```

Framework plugin interface:

```python
class FrameworkPlugin:

    def detect(self):
        """Return True if this plugin matches the current project."""
        pass

    def commands(self):
        """Return the framework-specific CLI commands this plugin exposes."""
        pass

    def extensions(self):
        """Return the PHP extensions this framework requires."""
        pass

    def services(self):
        """Return the recommended companion services (redis, mailpit, ...)."""
        pass
```
